Function 1: greet
    The greet function is designed to take the users name as a string input,
    then output a console message greeting the user.

Function 2: evenCheck
    The evenCheck function is designed to take a number as input, it then uses
    modulus to check if the remainder is 0. Using a conditional statement, it 
    returns a console message stating whether the inputted number is even or not.

Function 3: squareNumber
    The squareNumber function is designed to take a number as input, then square 
    the number and return the calculated value.

Function 4: farenToC
    The farenToC function is designed to take a number representing a measurement
    of degrees in farenheit, then roughly calculate a conversion into celcius. It
    then returns the calculated conversion.

Function 5: milesTraveled
    The milesTraveled function is designed to take two inputs: speed, and hours 
    traveled, respectively. It then does a basic calculation to determine how
    many miles were traveled, returning the output after.
